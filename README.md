# cavax

C language experimental utils for [Avalanche AVAX exploration](https://crypto.bi/category/avax/).

AKA playing with Avalanche AVAX in the C language.

## Build

Standard GNU `autotools` build procedure. See INSTALL.

## Run

    ./cavax
